POPUP
html 엘리먼트를 화면 중앙에 보여주는 기능
HtmlBinder 와 같이 사용하여 다른 Html의 내용을 불러오도록 사용하는것이 편리함
POPUP
여러 오브젝트를 생성할 필요가 없으므로 new 로 새 오브젝트를 생성하지 않고 POPUP 자체를 오브젝트로 사용함
.show(element)
#arg
element - 화면 중앙에 보여줄 엘리먼트
return : 없음
사용 예
POPUP.show($("<textarea>"));
.showWithURL(url,size)
#arg
url - htmlBinder 로 불러올 html 의 url
size - 팝업 사이즈 #기본값 {width:100,height:100} return : 없음
사용 예
POPUP.showWithURL("pages/sample1.html",{width:300,height:100});
.close()
팝업을 닫아주는 기능
사용 예
POPUP.close();
.setCoverCss(cssObj)
팝업이 오브젝트 외곽의 백그라운드 영역에 스타일을 강제로 지정
오브젝트 형태로 여러 스타일을 동시에 지정 가능함
#arg
cssObj - css 네임과 벨류의 값으로 이루어진 오브젝트
사용 예
POPUP.setCoverCss({
"background":"none"
});
특이사항
팝업이 열려있는중에 여러개를 여는경우 큐에 쌓이고 앞의 팝업이 닫히면 이어서 열림
엘리먼트 밖의 검은 영역을 터치하면 팝업 닫힘
