StyleSheetLoad
스타일시트를 불러오고 제거하는 기능
orchestra_lib.js 에서 함수로 제공
cssLoad(path,css_id)
css파일을 불러와서 헤더에 붙여주는 기능
동일한 아이디로 다른 css를 불러오면 기존에 불러온 css를 제거한다
#arg
path - css파일의 경로(절대경로로 설정시 웹서버를 통한 서비스라면 '/', 로컬이라면 /www/ 경로를 루트 경로로 인식함)
css_id - css 관리를 위한 아이디(기본값 "");
사용 예
cssLoad("/css/a.css","title");
cssLoad("/css/b.css","title");
cssRemove(css_id)
불러온 css를 제거한다
#arg
css_id - css 관리를 위한 아이디(기본값 "");
사용 예
cssRemove("title");
