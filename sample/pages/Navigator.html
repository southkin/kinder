Navigator
Html 엘리먼트를 child로 추가하여 정렬 순서에 따라 보여주는 위치를 변경하여 네비게이터 형태로 보여주는 기능
HtmlBinder 와 같이 사용하여 다른 Html의 내용을 불러오도록 사용하는것이 편리함
new Navigator(element)
arg : element - 하위 엘리먼트들을 네비게이션 형태로 보여줄 엘리먼트
return : Navigator 오브젝트
사용 예
var testNavi = new Navigator($("#testContainer"));
.push(element,description)
#arg
element - html엘리먼트
description - getCurrentDescription 함수 실행시 리턴시킬 현재 엘리먼트 뷰에 대한 설명 return : 없음
사용 예
#엘리먼트 생성
testNavi.push($("<div>").html('sample'));
#엘리먼트 참조
testNavi.push($("#testDiv"));
#HtmlBinder로 다른 Html의 내용을 불러온 엘리먼트 참조
var element = $("<div>",{style:"width:100%; height:100%; position:relative"});
var binder = new HtmlBinder(element);
binder.load("pages/sample1.html");
testNavi.push(element,'Navi_Child1');
.pushWithURL(url)
#arg
url - html의 URL
return : 없음
#전달받은 URL 의 HTML 내 BODY 를 HtmlBinder를 통해 불러와서 push 하는 함수로 description 값에는 url 이 저장된다.
사용 예
testNavi.pushWithURL('pages/sample1.html');
.getCurrentDescription()
#arg
없음
return : 현재 화면에 보여지는 엘리먼트의 description 값
사용 예
if (testNavi.getCurrentDescription().match(/\.(html|htm)/gi)) {
alert('html 파일 입니다.');
}
.back(cnt)
#arg
cnt - 뒤로 돌릴 갯수 //지정하지 않으면 기본값 1
return : 없음
사용 예
testNavi.back();
testNavi.back(1);
testNavi.back(9999);
.count()
#arg
없음
return : push 된 엘리먼트의 수
사용 예
var count = testNavi.count();
.hasBack()
#arg
없음
return : 뒤로 돌릴 수 있을정도로 push 된 엘리먼트가 있는지 여부(뷰 카운트가 2개 이상)
사용 예
if (testNavi.hasBack()) {
...
}
.goto(idx)
#arg
idx - 보여줄 엘리먼트의 인덱스 return : 없음
사용 예
testNavi.goto(1);
샘플

Navigator

이전
Class
[[NavigatorElement]]no_Animation
#push와 back 동작중 애니메이션을 사용하지 않음
Event
disappear
#화면에서 벗어날때 발생되는 이벤트
appear
#화면에서 벗어나있던 중 다시 화면에 보여질때 발생되는 이벤트
disable
#back 되어 사용되지 않게 되었을때 발생되는 이벤트
